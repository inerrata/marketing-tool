## 2. Search Intent

**Intent type:** Commercial investigation  
The searcher is comparing options before making a software decision. They are not yet ready to buy — they want a trusted shortlist with enough context to evaluate fit for their specific org type (size, budget, use case).

**Dominant content format (based on what ranks for this type of query):** Listicle/comparison ("Top 10 CRMs for Nonprofits in 2026"), often 1,500–2,500 words. Content must match this format to compete. A product page or a generic explainer will not rank here.

## 8. Differentiation Angle

The top-ranking pages for this keyword are often generic listicles that review the same 5–10 tools without genuine specificity. To beat them, choose one of the following angles and commit to it:

**Angle A — Segment by org type and budget (recommended):**  
Most nonprofit CRM content treats all nonprofits as identical. Real differentiation: build explicit decision paths ("If you're a 5-person org with a $0 tech budget, go here. If you're processing $2M+ in annual donations, go here."). This makes the piece more useful and more skimmable.

**Angle B — Real user data:**  
If you have access to nonprofit CRM reviews (G2, Capterra, Salesforce AppExchange), pull real user quotes and aggregate satisfaction scores to support each pick. This signals E-E-A-T and differentiates from competitors who write without evidence.

**Angle C — Current pricing + nonprofit discount accuracy:**  
Pricing changes constantly and most listicles are stale. A brief that commits to verifying current pricing (and noting last-verified date) provides concrete utility competitors don't.

Combining A + C is realistic without original data and meaningfully outperforms the average result.

## 11. E-E-A-T Signals to Include

This is a commercial-investigation piece in a category adjacent to financial decision-making (software procurement for organizations managing donor funds). Demonstrate credibility:

- **Author bio:** Include name, title, and 1-sentence credential (e.g., "X years covering nonprofit tech" or "former nonprofit operations director")
- **Last updated date:** Prominently shown at the top of the article — pricing and features change; stale content signals low trust
- **Sources:** Link to each tool's official pricing page; cite any statistics (e.g., "X% of nonprofits use CRM software") with a named source
- **First-hand signals:** If possible, include screenshots from actual tool trials or note if the tools were tested hands-on vs. desk-researched
- **Review methodology:** A brief 2–3 sentence note on how tools were selected and evaluated (criteria used, sources consulted)
